# 🛡️ Optimal Hedging Analysis

A quantitative finance project that demonstrates **optimal hedging techniques** for minimizing portfolio risk using Python. The project explores hedge ratio estimation, portfolio performance, and risk reduction through statistical and mathematical analysis. It serves as a practical introduction to derivative hedging and financial risk management.

---

## 📌 Overview

Financial markets are exposed to various sources of risk, including price fluctuations, volatility, and market uncertainty. This project analyzes how hedging strategies can be used to reduce portfolio risk by constructing an optimal hedge based on historical market data and quantitative methods.

The implementation focuses on evaluating hedge effectiveness using Python and provides a framework for experimenting with different market scenarios.

## 📊 Workflow

1. Load historical financial data.
2. Clean and preprocess the dataset.
3. Calculate asset returns.
4. Estimate volatility and covariance.
5. Compute the optimal hedge ratio.
6. Construct the hedged portfolio.
7. Compare hedged and unhedged performance.
8. Visualize results and analyze risk reduction.

## ▶️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/Optimal-Hedging-Analysis.git
```

Navigate to the project directory:

```bash
cd Optimal-Hedging-Analysis
```

Install the required libraries:

```bash
pip install pandas numpy matplotlib jupyter
```

## ▶️ Running the Project

Launch Jupyter Notebook:

```bash

```

Open:


Run all cells to perform the hedging analysis and visualize the results.
